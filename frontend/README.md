# THTools Frontend

React + Vite + Material-UI frontend for THTools application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Features

- **Home**: Welcome page with tool cards
- **TikTok Downloader**: Download videos from TikTok/Douyin
- **AI Generative**: AI content generation (placeholder)
- **Processing Video**: Video processing tools (placeholder)
- **Text To Speech**: Convert text to speech (placeholder)

## Tech Stack

- React 18
- Vite
- Material-UI
- React Router
- Axios

## Development

Frontend runs on: `http://localhost:5173`

API proxy configured to: `http://localhost:3000`

Make sure backend is running before starting frontend.
